1. # margin 설정 방법

   2. 각각 설정하는 방법   
   ```cs
      margin-top : 10px: 
      margin-left : 10px;
      margin-bottom : 10px;
      margin-right : 10px;
   ```   
      
   2. 상 / 우 / 하 / 좌     
   ```cs
      margin : 10px 20px 30px 40px;  /* 시계방향 */
   ```   
   상 - 10px;      
   우 - 20px;   
   하 - 30px;   
   좌 - 40px;   
      
   2. 상 / 좌우 / 하      
   ```cs
      margin : 10px 20px 30px;
   ```   
   상 - 10px;   
   좌우 - 20px;   
   하 - 30px;   

   2. 상하 / 좌우   
   ```cs 
      margin : 10px 20px;
   ```   
   상하 - 10px;   
   좌우 - 20px;
