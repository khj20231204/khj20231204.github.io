1. # &#39;(빽틱)을 사용한 블럭 코드는 들여쓰기 주의
&#39; &#39; &#39; java   
   system.out.println("hello");&#45;&#45;&#45;㉠   
   
   system.out.println("world");&#45;&#45;&#45;㉡   
&#39; &#39; &#39;  
블럭 코드 안에 소스들은 &#39;보다 들여쓰기를 해야 한다 
㉠과 ㉡ 사이에 공백을 두고 공백 &#39;(빽틱)이 그대로 화면에 나오고 태그가 적용되지 않는다.

$$$ 2. 순서1. 사용주의 $$$
1. # 제목1
1. # 제목2
   2. ## 소제목1
   2. ## 소제목2
1. # 제목3

toc에서 소제목을 인식한다.
1.aaaa
   1.bbbb
   2.cccc

2.dddd
이렇게 제목도 1. 2. 이렇게 나오고 소제목도 1. 2. 이렇게 나와서 보기 별로다.
하지만 새로 값을 사이에 입력시 그냥 2.으로 통일하면 된다.
//------------------------------

1. # 제목1
   1) __소제목1__

   2) __소제목__

1. # 제목2

toc에서 소제목을 인식 못 한다.
이렇게 하면 제목은 1. 2. 소제목은 1) 2) 이렇게 구별되는데 새로 값을 입력하면 숫자를 다 맞춰야하고 강조도 __ 로 직접해야 된다

//---------------------------
1. # 제목1
1. # 제목2
   <h3>1. 소제목</h3>
   <h3>2. 소제목</h3>
1. # 제목3
toc이 제대로 인식하는 건 제일 앞 1. 뿐이고 하위제목의 1.밑으론 제대로 인식 못한다
