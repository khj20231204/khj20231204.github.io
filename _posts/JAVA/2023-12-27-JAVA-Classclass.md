1. # Class 클래스
   자바의 모든 클래스와 인터페이스는 컴파일 후 class파일이 생성됩니다. Class 클래스는 이 파일에 대한 정보를 가져올 수 있습니다.   

1. # Class 객체 얻기   

   ### 클래스로부터 얻기   
   ```java
      Class cla = 클래스이름.class
      Class cla = Class.forName("패키지..클래스 이름")
   ```   

   ### 객체로부터 얻기   
   ```java
      Class cla = 참조변수.getClass();
   ```
