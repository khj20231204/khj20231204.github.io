1. # 컬렉션
   - Collection 계열   
   List계열 : 순서를 유지하고 저장, 중복 저장 가능   
   ex)ArrayList, Vector, LinkedList   
   
   Set계열 : 순서를 유지하지 않음, 중복 저장 안됨   
   ex)HashSet, TreeSet   
   
   - Map 계열   
   키와 값의 쌍으로 저장   
   키 중복 저장 안됨   
   ex)HashMap, HashTable, TreeMap, Properties
