1. # zoom 설치

   다운받기   
   ```
      wget https://zoom.us/client/latest/zoom_amd64.deb
   ```   

   설치   
   ```
      sudo apt install ./zoom_amd64.deb   
   ```   

   애플리케이션에서 zoom 검색하여 실행   


   *삭제 시   
   ```
      sudo apt remove zoom
   ```
